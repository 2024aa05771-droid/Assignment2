Problem statement :

In this assignment, you will be required to:

1. Implement multiple classification models

2. Develop an interactive Streamlit web application to showcase your models

3. Deploy the application on Streamlit Community Cloud (free)

4. Provide clickable links for evaluation

Through this task, you will gain hands-on experience with a real-world, end-to-end machine learning deployment workflow, including model development, evaluation, user interface design, and deployment.
